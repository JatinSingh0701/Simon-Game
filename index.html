Simon Game Rules

  • The game has four colored buttons: red, blue, green, and yellow.
  • The game randomly generates a sequence of button presses, starting with one button press and adding one more each round.
  • The player must repeat the sequence by pressing the buttons in the same order.
  • If the player enters the correct sequence, the game generates a new, longer sequence.
  • If the player enters the wrong sequence, the game ends and the player must start over.
  • The game keeps track of the player's score, which is the number of button presses in the longest sequence entered correctly.
  • The game continues until the player makes a mistake or quits.
